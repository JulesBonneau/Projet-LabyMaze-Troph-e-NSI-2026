# Projet-LabyMaze-Troph-e-NSI-2026
Notre projet Python est de créer un jeu en deux dimensions dans lequel le joueur joue un carré qui doit résoudre plusieurs labyrinthes, plus il gagne plus il obtient de pièces. Avec ces pièces le joueur peut acheter des packs avec lesquels il peut obtenir des personnages carrées encore plus rapide afin de résoudre les labyrinthes plus rapidement.
